Carousel Web Component example

Here's a list of images that uses a simple-carousel Web Component for progressive enhancement, converting a basic list of images into a carousel, if JavaScript works.

If you turn JavaScript off, you'll see a usable list of captioned images. With JavaScript, the same set of images is enhanced by the Web Component, transformed into a more compact and (somewhat) fancy carousel.

The carousel behavior and CSS are very basic, the goal is just to demonstrate how a Web Component can act as a progressive enhancement wrapper for a plain HTML structure that also works perfectly well by itself.
