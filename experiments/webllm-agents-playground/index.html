AI Agents playground

This is experimental, run at your own risk!
The results are meant as examples only, and as is common with large language models you might get surprising results, depending on which language model you select.
Those results are not meant to be used aynwhere except in this experiment.
Note also that this page downloads large model files to your computer, possibly several gigabytes, when you run the agents below.

This page is meant to help you understand the basics of how AI agents work. It runs 3 agents locally in your browser, using WebLLM.

The first run takes some time as the language model is downloaded in your browser. Once that's done, the example agents take a few seconds or even minutes to run, depending on the selected model and on how powerful your computer is.

The agents are defined below, in plain text that you can modify at will to experiment.

An input text, found below, is fed to the first agent, and other agents get the output of the previous one as their input.

To run this, your browser needs to support WebGPU. You'll get an error message if that's not the case, in the "status" field of the "output" section. Recent versions of Chrome work.

Look at the "output" section at the end of this page to see what's happening, as well as the results. The agents run automatically when the page is loaded, and you can run them again using the "run" button in that section of the page.
